Documentation

Guide complet d'installation et d'utilisation de SENTINEL.

Installation

macOS Intel (x86_64)

Mac Intel — modèles 2017 à 2020. Requiert Python 3.9+ et macOS Ventura 13+ recommandé.

# 1. Télécharger SENTINEL_Intel.zip
# 2. Ouvrir le guide install_guide.html
# 3. Contourner Gatekeeper (clic droit → Ouvrir)
# 4. Double-cliquer sur Sentinel.app

macOS Apple Silicon (arm64)

Puce Apple M1, M2, M3, M4 — 2020 et après. Launcher zsh avec PATH Homebrew ARM.

# Même procédure que Intel
# Le launcher détecte automatiquement l'architecture

Windows 10 / 11

Requiert Python 3.9+ avec "Add Python to PATH" coché lors de l'installation.

# 1. Installer Python : python.org/downloads
#    Cocher impérativement "Add Python to PATH"
# 2. Extraire SENTINEL_Windows.zip
# 3. Double-cliquer sur SENTINEL.bat

Linux — Ubuntu / Debian / Fedora

# Ubuntu/Debian
sudo apt install python3 lsblk smartmontools lm-sensors

# Fedora
sudo dnf install python3 util-linux smartmontools lm_sensors

# Lancer SENTINEL
chmod +x sentinel.sh
./sentinel.sh

Fonctionnement

SENTINEL collecte les informations système via les commandes natives de chaque OS, génère un rapport HTML et l'ouvre dans le navigateur. L'analyse dure entre 30 et 90 secondes.

Modes d'affichage

Mode Simple

Conçu pour les utilisateurs non-techniques. Chaque information est résumée en une carte avec un verdict coloré (vert / orange / rouge) et une action corrective si nécessaire.

Mode Expert

Toutes les métriques techniques : températures, tensions, IOPS, SMART, logs de démarrage, connexions réseau actives, permissions par application.

Confidentialité

SENTINEL ne contacte aucun serveur externe. Toutes les analyses sont locales. Le rapport HTML reste sur votre machine. Votre mot de passe n'est jamais stocké sur disque.

Version automatique

La version de SENTINEL suit le format M.J (mois.jour). La version actuelle est v3.27. Elle se met à jour automatiquement à chaque chargement de page.
