À propos de
SENTINEL

SENTINEL est une application de diagnostic système multi-plateforme conçue pour rendre les informations techniques accessibles à tous.

Le projet

Développé par SHAMS&DEAN, SENTINEL analyse votre ordinateur et génère un rapport HTML 100% local. Aucune donnée n'est transmise à un serveur externe.

Deux modes : Mode Simple avec verdicts colorés et actions concrètes — Mode Expert avec toutes les métriques détaillées.

Technologies

Python 3.9+, compatible macOS (Intel et Apple Silicon), Windows 10/11 et Linux. Le rapport est un fichier HTML autonome s'ouvrant dans n'importe quel navigateur.

Ce site est un fichier HTML unique avec navigation sans rechargement, curseur animé bicercle, mode sombre via data-theme et articles chargés via fetch().
