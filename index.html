À propos
de SENTINEL

Application de diagnostic système multi-plateforme conçue pour rendre les informations techniques accessibles à tous, des débutants aux experts.

Le projet

Développé par SHAMS&DEAN, SENTINEL analyse votre ordinateur et génère un rapport HTML 100% local. Aucune donnée n'est transmise à un serveur externe. Le rapport s'ouvre dans n'importe quel navigateur.

Deux modes d'affichage : Mode Simple pour les utilisateurs non-techniques — Mode Expert avec toutes les métriques détaillées pour les professionnels.

Technologies

Python 3.9+, compatible macOS (Intel et Apple Silicon), Windows 10/11 et Linux. Ce site est un fichier HTML unique avec navigation sans rechargement, curseur animé, mode sombre et articles chargés via fetch().
