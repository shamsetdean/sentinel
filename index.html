SENTINEL — développé
par SHAMS&DEAN

SENTINEL analyse votre ordinateur et génère un rapport HTML complet 100% local. Aucune donnée n'est transmise à un serveur externe. Le rapport s'ouvre dans n'importe quel navigateur.

Deux modes d'affichage : Mode Simple pour les utilisateurs non-techniques — verdicts colorés et actions concrètes. Mode Expert avec toutes les métriques brutes pour les professionnels IT.

LinkedIn X GitHub Notion

SimpleMode Simple — pour tout le monde

Cartes compactes avec verdicts colorés. Vert = bon, Orange = surveiller, Rouge = agir. Lecture complète en moins de 30 secondes, sans jargon technique.

ExpertMode Expert — pour les professionnels

240+ métriques, températures, tensions, IOPS, SMART, LaunchAgents, connexions réseau actives. Pour les administrateurs système et professionnels IT.
