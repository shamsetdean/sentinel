## Versionnement automatique

La version suit le format `M.J` (mois.jour du jour de consultation).  
Elle est calculée automatiquement en JavaScript à chaque chargement de page — aucune mise à jour manuelle requise.

```javascript
// Calcul automatique de la version
var d = new Date();
var version = d.getMonth() + 1 + '.' + d.getDate();
// Aujourd'hui : 3.27
```
